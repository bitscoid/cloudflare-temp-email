# Frontend

Vue 3 frontend for Cloudflare Temp Email.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

```bash
pnpm deploy
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `VITE_API_BASE` | Worker API URL |
| `VITE_IS_TELEGRAM` | Telegram mini app mode |

## Tech Stack

- Vue 3
- Naive UI
- Vue Router
- Vue I18n
- Axios
- Vite
